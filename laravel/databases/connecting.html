# Configuratie via .env

Een .env-file is een bestand waarin je de configuratie van je applicatie kan opslaan. Het voordeel hiervan is dat je deze configuratie niet in je code hoeft te zetten. Commit een .env-file nooit naar je repository.

Je kan een .env-file aanmaken door een kopie te maken van het bestand .env.example en deze te hernoemen naar .env.

Laravel maakt gebruik van de .env bestand om de database configuratie op te slaan. In dit bestand kan je de database configuratie aanpassen. De standaard configuratie ziet er als volgt uit:

DB_CONNECTION=mysql
DB_HOST=
DB_PORT=
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
